VluxySF

This item only works when running on the client. ClientThis item only works when running on the server. Server

VluxySoundFactory is a sound management system designed to simplify the process of dealing with sounds as a scripter. It primarily tries to make fetching your sounds as easy as possible and comes with some memory saving features. It also comes with some extra features like jittering and presets.

Fetch​

RequiresSchema
</>
VluxySF.Fetch(
name: string,--

The name of the Sound.

parent: Instance?,--

Parent of the sound.

preset: (Sound) → ()?--

Preset to apply.

) → Sound--

The fetched sound instance.

Finds the specified SoundDefinition and gives back a Sound. If somthing goes wrong a Blank Sound is returned instead.

INFO
  • If a sound is Preloaded it is instead Cloned from the preloadFolder
  • If a sound is Warmed it is instead Cloned from the warmed cache.
WARNING

If the sound is not preloaded or warmed it may take a moment to fetch and load as it needs to be created first.

local mySound = VluxySF.Fetch("MySound")
mySound.Volume = 0.3

local echoEffect = mySound.EchoSoundEffect
if echoEffect then
	echoEffect.WetLevel = 1.3
end

mySound:Play()

Parent of the Sound.

preset: (Sound) → ()?--

Preset to apply.

) → Sound?--

The fetched Sound instance if success.

Similar to VluxySF.Fetch but will not return a Blank Sound on fail.

local mySound = VluxySF.Fetch("MySound")

if mySound then
	mySound.Volume = 0.3

	local echoEffect = mySound.EchoSoundEffect
	if echoEffect then
		echoEffect.WetLevel = 1.3
	end

	mySound:Play()
end
